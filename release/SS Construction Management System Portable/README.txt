SS Construction Management System
Portable Windows Release

Contents
- SS Construction Management System.exe
- data\
- exports\

How to use
1. Extract this folder on a Windows laptop or desktop.
2. Open "SS Construction Management System.exe".
3. The application will save its database inside the local data folder.
4. Exported PDF reports will be created inside the local exports folder.

Recommended setup
- Keep the whole folder together.
- Do not run the EXE directly from inside a ZIP file.
- If Windows SmartScreen appears, choose More info and then Run anyway if you trust this build.

Moving to another laptop
- Copy the full folder, including data and exports, to keep your records.
- If you want a fresh start, copy only the EXE.
